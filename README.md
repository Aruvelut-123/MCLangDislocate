# MCLangDislocate
一个自动打乱MC语言文件的python程序（包括材质包和模组）  
目前仅支持打乱中文  
老版本使用lang格式的MC暂时不支持
## 使用方法
1. 克隆此仓库  
在命令行中运行`git clone https://github.com/Aruvelut-123/MCLangDislocate.git`
2. 安装依赖  
   在命令行中运行`pip install -r requirements.txt`
3. 运行程序  
   在命令行中运行`python main.py`
4. 根据提示操作
5. 程序会自动将打乱后的文件打包为材质包生成到根目录
6. 将材质包放入游戏并加载即可
